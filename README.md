# Design
## Operators
- +,-,/,*
- =
- ==, !=, >, <, >=,<=
- and, or, not
## Variables
```
var a = 0
var a = "hi"
```
## Ifs
```
if a==2 then
   print("hi")
end
```
```
if a==2 then
   print("2")
end
else if a==3 then
   print("3")
end
else then
   print(".....")
end
```
## While
```
while a==2 do
   print("hi")
end
```
